Цель работы
Освоить базовые типы данных в языке C, изучить взаимосвязь переменных и выделяемой памяти, научиться использовать операции и выражения, организовать ввод и вывод данных, а также разобраться с эффектами переполнения и ограничениями точности вычислений.

Используемые технологии
Язык программирования: C

Компилятор: gcc (или MinGW)

Операционная система: Windows

Порядок запуска
Открыть командную строку (cmd) или терминал

Перейти в папку с заданием: cd C:\lab01

Скомпилировать нужный файл:
gcc task1.c -o task1.exe

Выполнить программу:
task1.exe

Перечень выполненных заданий
Первая программа с использованием переменных

Определение размеров типов данных

Работа с переменными и их расположением в памяти

Вычисление остатка от деления

Операции инкремента и декремента

Явное и неявное преобразование типов

Потеря точности при вычислениях

Эффект переполнения
